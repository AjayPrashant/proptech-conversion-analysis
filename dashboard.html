Customer psychology — why they don't open
Overchoice effect
Analysis paralysis
7–9 documents presented with equal visual weight. When inputs are indistinguishable in importance, investors defer rather than choose the wrong starting point. Deferral equals abandonment.
Dunning-Kruger exit
Competence gap shock
First-timers arrive believing they can evaluate investment documents. A 40-page Investment Memorandum immediately reveals the gap. They exit — not from disinterest, but from rational self-assessment.
Trust architecture
The document paradox
Documents simultaneously signal legitimacy and create overwhelm. Returning investors use them as confirmation. First-timers need them as foundation — but the format was built for the wrong audience.
Context mismatch
Mobile PDF problem
59% of users are mobile. PDFs require active zoom and scroll — high-effort interaction that raises cognitive load, triggers doubt, and accelerates exit. The experience was designed for desktop and used on mobile.
05
Recommendation — reduce activation energy
Phase 1 — Now · Zero compliance risk · No legal review required
Guided document hierarchy
Reorder, label, and contextualise the document list so investors know where to start. No financial summaries generated, no AI involved — metadata-only labels on existing documents. Can ship within a single sprint.
Surface one document first with a clear "Start Here" prompt — the data shows any first document opens the conversion pathway, not a specific one
Add a one-line plain-English label to each document: "What is this file?" — not a summary, just context
Mobile-optimised reading mode — chunked, scrollable, no PDF viewer dependency for mobile users (59% of traffic)
"Key Numbers" card using 4–5 structured fields already in the database (yield, ROI, occupancy assumption, exit timeline)
Phase 2 — Q2 · Requires data pipeline
Structured data layer
Extract key metrics from documents into structured fields. Enables cross-listing comparison, yield calculator with investor's own capital input. No compliance exposure — structured data only.
Phase 3 — Q3+ · Compliance review required
AI document Q&A
Plain-language investor questions answered by AI trained on document content. Highest potential impact. Requires legal sign-off on response boundaries and disclaimers before any build begins.
